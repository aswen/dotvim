Vim config
===========

Installation
------------

```bash
cd ~
git clone git://github.com/aswen/dotvim.git
cd dotvim
./createlinks
```

Adding modules
--------------

```bash
cd ~/dotvim
git submodule add <giturl> bundle/<modulename>
```
